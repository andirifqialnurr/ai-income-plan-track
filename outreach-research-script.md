# Day 1 - Script Riset Prospek

Tujuan file ini: memvalidasi prospek dari `prospects_mobile_detailers.csv` sebelum mereka masuk batch outreach.

## Riset 3 Menit Per Prospek

Untuk setiap prospek, cek:

1. Website masih aktif.
2. Mereka benar-benar melayani mobile detailing.
3. Ada tombol quote, booking, contact, SMS, email, atau phone.
4. Ada layanan bernilai tinggi: full detail, ceramic coating, paint correction, fleet, maintenance plan.
5. Ada tanda mereka masih aktif: review baru, galeri terbaru, Instagram aktif, booking page aktif.

## Kolom Yang Perlu Diisi Setelah Verifikasi

Tambahkan manual di CSV kalau sudah dicek:

- `owner_name` jika ditemukan.
- `instagram_url` jika ditemukan.
- `contact_form_url` jika ada.
- `personalization_note` seperti "punya package ceramic coating" atau "menekankan fast replies".
- `outreach_status`: not_contacted, contacted, replied, demo_sent, call_booked, closed_won, closed_lost.

## Angle Pesan

Angle utama:

> You already get quote requests. I help you answer them faster, ask the missing details, and keep every lead organized so fewer inquiries go cold.

Bahasa outreach yang dipakai tetap bahasa Inggris karena target pasar AS.

```text

```

## Jawaban Kalau Mereka Tanya "What does it do?"

```text
It connects to your quote form or a simple new form.

When someone asks for a detail, it can:
1. reply instantly,
2. ask missing questions,
3. summarize the job for you,
4. store the lead in a simple dashboard,
5. trigger follow-ups if they do not book.

The point is not to replace you. It stops warm leads from going cold while you are detailing cars.
```

## Jawaban Kalau Mereka Tanya Harga

```text
Founding install is $349.

That includes setup, your branded quote form, AI reply logic, lead dashboard, and 3 follow-up messages.

After the first month, ongoing maintenance is optional at $79/month.
```

## Kriteria Siap Outreach

Prospek siap dikontak kalau:

- Website aktif.
- Mobile service jelas.
- Kontak ditemukan.
- Ada personalisasi minimal 1 kalimat.
